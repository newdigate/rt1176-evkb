# RT1176 `I2S` audio out (SAI1 TX, polled) + WM8962 codec — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Transmit a known 1 kHz sine (48 kHz / 16-bit / stereo I²S) out SAI1 on the MIMXRT1176-EVKB through the WM8962 codec, verified sample-exact in QEMU and audible/Saleae-verified on hardware.

**Architecture:** A polled/blocking `I2S` core class (SPI/Wire-style `hardware_t` singleton) drives SAI1 as I²S master; a `WM8962Codec` helper configures the codec over a new `Wire2`=LPI2C5. QEMU already models the SAI device — we add a deterministic **raw TDR-sample tap** (for exact data capture) and a **framing-agnostic WM8962 I²C-slave stub** (so the identical codec-init sketch runs in-sim). The audio PLL is brought up via the ANATOP AI indirect interface.

**Tech Stack:** ARM GCC 10.2.1 (`ARMGCC_DIR=/Applications/ARM_10`), CMake + teensy-cmake-macros, QEMU `mimxrt1170-evk` (`~/Development/qemu2`, `ninja`), LinkServer, Saleae Logic 2.

## Confirmed constants (all cited to SDK cm7 header / fsl_* / QEMU model)

**SAI1** base `0x40404000`, IRQ **76**. Offsets: TCSR `0x08`, TCR1 `0x0C`, TCR2 `0x10`, TCR3 `0x14`, TCR4 `0x18`, TCR5 `0x1C`, TDR0 `0x20`, TFR0 `0x40`, TMR `0x60`.
- TCSR: TE `1<<31`, BCE `1<<28`, FRF `1<<16`, FWF `1<<17`, FEF `1<<18`, SR `1<<24`, FR `1<<25`.
- TCR2: DIV `[7:0]`, BCD `1<<24`, BCP `1<<25`, MSEL `[27:26]`. TCR3: TCE `[19:16]`.
- TCR4: FSD `1<<0`, FSP `1<<1`, FSE `1<<3`, MF `1<<4`, SYWD `[12:8]`, FRSZ `[20:16]`. TCR5: FBT `[12:8]`, W0W `[20:16]`, WNW `[28:24]`.

**Clocks** (CCM root n @ `0x40CC0000+n·0x80`; LPCG n @ `0x40CC6000+n·0x20`):
- SAI1 root = idx **64** → `CCM_CLOCK_ROOT64_CONTROL 0x40CC2000` (mux=4 AudioPLL, div=16 → 24.576 MHz).
- SAI1 LPCG = idx **123** → `CCM_LPCG123_DIRECT 0x40CC6F60`.
- LPI2C5 root = idx **41** → `CCM_CLOCK_ROOT41_CONTROL 0x40CC1480` (mux=1, div=0 → 24 MHz).
- LPI2C5 LPCG = idx **102** → `CCM_LPCG102_DIRECT 0x40CC6CC0`.
- Audio PLL (ANADIG base `0x40C84000`): PLL_AUDIO_CTRL `0x40C84300` (ENABLE_CLK `1<<13`, GATE `1<<14`, STABLE `1<<29`). AI: CTRL_AUDIO `0x40C84880`, WDATA_AUDIO `0x40C84890`, RDATA_AUDIO `0x40C848A0` (AITOGGLE `1<<8`, AITOGGLE_DONE `1<<9`, AIRWB `1<<16`, AI_ADDR `[7:0]`). AI reg addrs: CTRL0 `0x00`, CTRL0_SET `0x04`, CTRL0_CLR `0x08`, CTRL2(num) `0x20`, CTRL3(denom) `0x30`. Config {loopDiv=32, num=768, den=1000, postDiv=1(÷2)} → 393.216 MHz.

**SAI1 pins** (all ALT0, pad-ctl value `0x02`; TX are outputs → **no SELECT_INPUT/daisy**):
- MCLK `GPIO_AD_17`: mux `0x400E8150`, pad `0x400E8394`. DATA00 `GPIO_AD_21`: mux `0x400E8160`, pad `0x400E83A4`.
- TX_BCLK `GPIO_AD_22`: mux `0x400E8164`, pad `0x400E83A8`. TX_SYNC `GPIO_AD_23`: mux `0x400E8168`, pad `0x400E83AC`.
- `IOMUXC_GPR_GPR0 0x400E4000`, SAI1_MCLK_DIR = `1<<8` → set 1 (MCLK output).

**LPI2C5** base `0x40C34000` (LPSR domain), IRQ **36**. Master regs (LPI2C1 offset pattern): MCR `+0x10`, MSR `+0x14`, MIER `+0x18`, MCFGR1 `+0x24`, MCCR0 `+0x48`, MFCR `+0x58`, MTDR `+0x60`, MRDR `+0x70`. Slave regs at `+0x110…` (same pattern as LPI2C1/2).
- Pins (LPSR IOMUXC base `0x40C08000`, ALT0, pad-ctl `0x0A`): SCL `GPIO_LPSR_05` mux `0x40C08014`, pad `0x40C08054`, `LPI2C5_SCL_SELECT_INPUT 0x40C08084` daisy 0; SDA `GPIO_LPSR_04` mux `0x40C08010`, pad `0x40C08050`, `LPI2C5_SDA_SELECT_INPUT 0x40C08088` daisy 0.

**WM8962** addr `0x1A`. Wire framing (fsl_wm8962.c:917 + fsl_codec_i2c.c:71): write reg R val V → on-wire bytes `[0x00, R, (V>>8)&0xFF, V&0xFF]` (2-byte big-endian subaddress + byte-swapped 16-bit data). Init read-polls reg `0x5D` (bit0=0=done) 3× and reads `0x04` once.

    Serial1.begin(115200);
    delay(50);
    I2S.begin(48000);
    // Stage A: verify SAI1 configured as 48k/16-bit I2S master.
    uint32_t tcsr = SAI1_TCSR, tcr2 = SAI1_TCR2, tcr4 = SAI1_TCR4, tcr5 = SAI1_TCR5;
    bool te   = (tcsr & SAI_TCSR_TE) != 0;
    bool div7 = (tcr2 & 0xFFu) == 7;
    bool mclk1= ((tcr2 >> 26) & 0x3u) == 1;
    bool i2s4 = (tcr4 & SAI_TCR4_MF) && (tcr4 & SAI_TCR4_FSD) &&

static inline void ai_udelay(volatile uint32_t n) { while (n--) __asm__ volatile("nop"); }

/* One ANATOP AI indirect write to the Audio PLL register file. Matches the
 * SDK ANATOP_AI_Write (fsl_anatop_ai.c:103) and the QEMU imxrt_anadig AI model
 * (toggle bit8 -> done bit9). */
static void ai_write(uint8_t aiAddr, uint32_t wdata) {
    uint32_t pre = ANADIG_MISC_AI_CTRL_AUDIO & ANADIG_AI_TOGGLE_DONE;
    ANADIG_MISC_AI_CTRL_AUDIO &= ~ANADIG_AI_RWB;                 // write mode
    ANADIG_MISC_AI_CTRL_AUDIO = (ANADIG_MISC_AI_CTRL_AUDIO & ~0xFFu) | aiAddr;
    ANADIG_MISC_AI_WDATA_AUDIO = wdata;
    ANADIG_MISC_AI_CTRL_AUDIO ^= ANADIG_AI_TOGGLE;              // request
    while ((ANADIG_MISC_AI_CTRL_AUDIO & ANADIG_AI_TOGGLE_DONE) == pre) { }
}

/* AI register sub-addresses (fsl_anatop_ai.h kAI_PLLAUDIO_*): CTRL0=0x00,
 * CTRL0_SET=0x04, CTRL0_CLR=0x08, CTRL2(num)=0x20, CTRL3(denom)=0x30.
 * CTRL0 bits: loopDiv[6:0], HOLD_RING_OFF b13, POWER_UP b14, ENABLE b15,
 * BYPASS b16, PLL_REG_EN b22, POST_DIV[27:25].  EXACT bit values/sub-addresses
 * must be transcribed from fsl_clock.c:749-798 (CLOCK_InitAudioPll) +
 * fsl_clock.h:1843 — the values below are the confirmed encoding for
 * {loopDiv=32, num=768, den=1000, postDiv=1 -> ÷2 -> 393.216 MHz}. */
static void sai1_audio_pll_init(void) {
    ai_write(0x04, (1u << 16));                 // CTRL0_SET: BYPASS
    ANADIG_PLL_AUDIO_CTRL |= ANADIG_PLL_AUDIO_CTRL_ENABLE_CLK;
    ai_write(0x30, 1000);                       // denominator
    ai_write(0x20, 768);                        // numerator
    /* loopDiv=32 into CTRL0[6:0], postDiv=1 into CTRL0[27:25] (read-modify via SET/CLR) */
    ai_write(0x08, 0x7Fu);                      // CTRL0_CLR: clear loopDiv field
    ai_write(0x04, 32u & 0x7Fu);                // CTRL0_SET: loopDiv=32
    ai_write(0x08, 0x7u << 25);                 // CTRL0_CLR: clear postDiv field
    ai_write(0x04, (1u << 25));                 // CTRL0_SET: postDiv=1 (÷2)
    ai_write(0x04, (1u << 22));                 // CTRL0_SET: PLL_REG_EN
    ai_udelay(20000);                           // >=100us settle

    ai_write(0x04, (1u << 15));                 // CTRL0_SET: ENABLE (clk out)
    ANADIG_PLL_AUDIO_CTRL &= ~ANADIG_PLL_AUDIO_CTRL_GATE;  // ungate
    ai_write(0x08, (1u << 16));                 // CTRL0_CLR: clear BYPASS
}

    hw->tcsr = SAI_TCSR_SR; hw->tcsr = 0u;
    hw->tcsr = SAI_TCSR_FR; hw->tcsr = 0u;
    hw->tmr = 0u;
