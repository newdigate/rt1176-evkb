# RT1176 SAI1 RX (Audio Capture) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add SAI1 receive (audio capture) to the imxrt1176 core — `read()` (polled) + `beginReceiveDMA()` (source 54), full-duplex with the existing TX — proven in QEMU via an RX injector, then on silicon with the onboard mic + a 1 kHz acoustic loopback.

**Architecture:** The RX registers/bits mirror TX (base `0x40404000`, RX block at `+0x88`). RX runs **synchronous to TX** (`RCR2.SYNC=1`), sharing the TX-generated BCLK/FS, so only the `SAI1_RXD[0]`=`GPIO_AD_20` input pin is new. `configureSAI()` becomes idempotent (a `configured_` guard) and all TX/RX enables use `|=`, so `write`/`read`/`beginDMA`/`beginReceiveDMA` compose for full-duplex without clobbering. QEMU gains an `rx-inject` chardev + `rx_ring` + a private `rx_timer` mirroring the TX tap/timer (TX path untouched).

**Tech Stack:** C++ core, Python generator, CMake + `arm-none-eabi` cross toolchain (`/Applications/ARM_10/bin/`), QEMU `mimxrt1170-evk` gates via `tools/qrun`. Reuses `DMAChannel` (eDMA) + DMAMUX source 54 + the WM8962 record path (already configured).

**Spec:** `docs/superpowers/specs/2026-07-06-rt1176-sai-rx-design.md`.

**Repos** (commit each in its own): core files `cores/imxrt1176/*` → nested **teensy-cores** repo (`~/Development/rt1170/evkb/cores`); QEMU → **qemu-rt1170** gitlab (`~/Development/qemu2`); gate `evkb/sai_rx_test/*` → outer **evkb** repo (local-only, no push). Commit to `master`.

- [ ] **Step 2: Add the RX tick + chardev receive handlers**

In `hw/audio/imxrt_sai.c`, add these static functions immediately **before** `imxrt_sai_realize`:

```c
/* Frame-clock for the receive side: pace the RX FIFO DMA request while RE is
 * set and injected samples are queued.  Mirror of imxrt_sai_tx_tick; the TX
 * timer is left completely untouched. */
static void imxrt_sai_rx_tick(void *opaque)
{
    IMXRTSAIState *s = opaque;

    if (!(s->regs[SAI_RCSR / 4] & CSR_EN)) {
        return;                 /* receiver disabled; leave the timer idle */
    }
    if (s->rx_level > 0) {
        qemu_set_irq(s->dma_rx, 1);
        qemu_set_irq(s->dma_rx, 0);
    }
    timer_mod(s->rx_timer, qemu_clock_get_ns(QEMU_CLOCK_VIRTUAL) +
                           imxrt_sai_tx_period_ns(s));
}

/* rx-inject chardev: bytes streamed in (LE int16) queue into rx_ring, which the
 * guest drains via RDR reads (polled) or the eDMA (dma_rx). */
static int imxrt_sai_rx_can_receive(void *opaque)
{
    IMXRTSAIState *s = opaque;
    return (int)((IMXRT_SAI_TX_SAMPLES - s->rx_level) * sizeof(int16_t));
}

static void imxrt_sai_rx_receive(void *opaque, const uint8_t *buf, int size)
{
    IMXRTSAIState *s = opaque;
    for (int i = 0; i + 1 < size; i += 2) {
        if (s->rx_level >= IMXRT_SAI_TX_SAMPLES) {
            break;
        }
        s->rx_ring[s->rx_head] = (int16_t)(buf[i] | (buf[i + 1] << 8));
        s->rx_head = (s->rx_head + 1) % IMXRT_SAI_TX_SAMPLES;
        s->rx_level++;
    }
}
```

- [ ] **Step 4: RCSR write starts/stops the RX frame clock**

```c
    case SAI_RCSR: {
        uint32_t old = s->regs[SAI_RCSR / 4];
        uint32_t new = value & ~(CSR_SR | CSR_FR);

        s->regs[SAI_RCSR / 4] = new;

            }
        }

        return;
    }
```

```cpp
void I2SClass::read(int16_t *s, size_t n) {
    hw->rcsr |= SAI_RCSR_RE;                      // enable the receiver (idempotent)
    for (size_t i = 0; i < n; i++) {
        uint32_t g = 2000000;                    // data-available guard (bounded)

}
```

```cpp
static DMAChannel i2s_rx_dma(false);
static int16_t *i2s_rx_ring;
static size_t i2s_rx_frames;
static void (*i2s_rx_onblock)(int16_t *);
static volatile uint32_t i2s_rx_blocks;

static void i2s_rx_isr() {
    i2s_rx_dma.clearInterrupt();
    i2s_rx_blocks++;
    if (i2s_rx_onblock) {
        uint32_t daddr    = (uint32_t)i2s_rx_dma.TCD->DADDR;
        uint32_t halfByte = (uint32_t)(i2s_rx_frames * 2u * sizeof(int16_t) / 2u);
        if (daddr < (uint32_t)i2s_rx_ring + halfByte)
            i2s_rx_onblock(i2s_rx_ring + i2s_rx_frames);   // 2nd half just filled
        else
            i2s_rx_onblock(i2s_rx_ring);                   // 1st half just filled
    }
}

bool I2SClass::beginReceiveDMA(int16_t *ring, size_t nFrames, void (*onBlock)(int16_t *)) {
    configureSAI();                              // idempotent
    i2s_rx_ring = ring; i2s_rx_frames = nFrames; i2s_rx_onblock = onBlock; i2s_rx_blocks = 0;
    i2s_rx_dma.begin();
    if (i2s_rx_dma.TCD == 0) return false;       // no channel free
    i2s_rx_dma.source(*(volatile uint16_t *)&SAI1_RDR0);                 // fixed FIFO source
    i2s_rx_dma.destinationBuffer((uint16_t *)ring, (unsigned int)(nFrames * 2u * sizeof(int16_t)));
    i2s_rx_dma.interruptAtHalf();
    i2s_rx_dma.interruptAtCompletion();
    i2s_rx_dma.triggerAtHardwareEvent(DMAMUX_SOURCE_SAI1_RX);
    i2s_rx_dma.attachInterrupt(i2s_rx_isr);
    i2s_rx_dma.enable();
    hw->tcsr |= SAI_TCSR_TE | SAI_TCSR_BCE;      // ensure the shared clock is running
    hw->rcsr |= SAI_RCSR_RE | SAI_RCSR_FRDE;     // RX + FIFO DMA request
    return true;
}
uint32_t I2SClass::rxBlockCount() const { return i2s_rx_blocks; }
```
