# RT1176 eDMA Driver (DMAChannel) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Port a faithful Teensy `DMAChannel` to the imxrt1176 core over the RT1176's classic eDMA + DMAMUX, and prove it by converting I²S TX from polled to DMA-fed (glitch-free sine, CPU idle).

**Architecture:** The RT1176 has the *same* classic eDMA (32 ch, 32-byte TCDs) + DMAMUX as Teensy 4 (RT1062), relocated to eDMA `0x40070000` / DMAMUX `0x40074000`. So `cores/teensy4/DMAChannel.{h,cpp}` ports with only a base-address retarget, backed by a generator-emitted register block. A `beginDMA()` seam on `I2SClass` wires a circular ring → `SAI1_TDR0`, paced by the SAI TX FIFO DMA request (DMAMUX source 55). Proven mem-to-mem then DMA-fed-audio in QEMU (which already models the whole SAI→DMAMUX→eDMA chain and accepts 16-bit TDR writes), then audible on silicon.

**Tech Stack:** C++ (Arduino/Teensyduino core), Python generator (`gen_imxrt1176_h.py`), CMake + `arm-none-eabi` cross toolchain, QEMU `mimxrt1170-evk` gates via `tools/qrun`. **No qemu2 changes** — the eDMA/DMAMUX models, SoC wiring, and SAI tap all already exist. All work is under `cores/imxrt1176/` + a new `evkb/edma_test/` gate. Commit to `master` (project convention).

**Spec:** `docs/superpowers/specs/2026-07-05-rt1176-edma-driver-design.md`.

**Reference (read-only, in-tree):** `cores/teensy4/DMAChannel.h` (592 lines), `cores/teensy4/DMAChannel.cpp` (147 lines), `cores/teensy4/imxrt.h` (DMA/DMAMUX macros). The `evkb/i2s_audio_test/` gate is the pattern to mirror.

**Key facts (verified):** eDMA `0x40070000`, TCD block `0x40071000` (ch stride `0x20`), DMAMUX `0x40074000` (CHCFGn stride 4). DMA channel IRQs = NVIC 0..15, error = 16. eDMA clock = CCM LPCG 22 (`CCM_LPCG22_DIRECT @ 0x40CC62C0`). `DMAMUX_SOURCE_SAI1_TX = 55` (matches silicon `PERI_DMAMUX.h` *and* QEMU). `NVIC_ENABLE_IRQ`/`_DISABLE_IRQ`/`_SET_PRIORITY` in `imxrt1176.h:154-158`; `_VectorsRam` in `core_pins.h:65`; `__disable_irq` in `core_pins.h:73`. `SAI1_TDR0 @ 0x40404020`. `import_arduino_library` uses `file(GLOB)` **without** `CONFIGURE_DEPENDS`, so **`rm -rf build`** after adding core files.

```cpp
#include <Arduino.h>
#include "HardwareSerial.h"
#include "DMAChannel.h"

// ---- Stage A: software-triggered memory-to-memory (buffers in OCRAM) ----
static DMAMEM uint32_t a_src[16];
static DMAMEM uint32_t a_dst[16];
static DMAChannel a_dma(false);          // (false) = don't allocate at static init
static volatile uint32_t a_done;
static void a_isr() { a_dma.clearInterrupt(); a_done = 1; }

void setup() {
    Serial1.begin(115200);
    delay(50);

    for (int i = 0; i < 16; i++) { a_src[i] = 0xA5A50000u + i; a_dst[i] = 0; }
    a_dma.begin();                        // allocate a channel, init TCD
    a_dma.sourceBuffer(a_src, sizeof(a_src));
    a_dma.destinationBuffer(a_dst, sizeof(a_dst));
    a_dma.interruptAtCompletion();
    a_dma.attachInterrupt(a_isr);
    a_dma.triggerManual();                // SSRT -> START: runs the whole major loop
    uint32_t guard = 2000000;
    while (!a_done && guard--) { }
    bool ok = a_done;
    for (int i = 0; i < 16; i++) if (a_dst[i] != a_src[i]) ok = false;
    Serial1.println(ok ? "STAGE_A_PASS" : "STAGE_A_FAIL");
}

void loop() { }
```

Then add the DMA path (file-scope statics + methods) at the end of `I2S.cpp`:

```cpp
static DMAChannel i2s_dma(false);
static const int16_t *i2s_ring;
static size_t i2s_ring_frames;
static void (*i2s_refill)(int16_t *);
static volatile uint32_t i2s_blocks;

static void i2s_dma_isr() {
    i2s_dma.clearInterrupt();
    i2s_blocks++;
    if (i2s_refill) {
        // Which half just drained? SADDR has advanced past it.
        uint32_t saddr    = (uint32_t)i2s_dma.TCD->SADDR;
        uint32_t halfByte = (uint32_t)(i2s_ring_frames * 2u * sizeof(int16_t) / 2u);
        if (saddr < (uint32_t)i2s_ring + halfByte)
            i2s_refill((int16_t *)i2s_ring + i2s_ring_frames);   // 2nd half drained
        else
            i2s_refill((int16_t *)i2s_ring);                     // 1st half drained
    }
}

bool I2SClass::beginDMA(const int16_t *ring, size_t nFrames, void (*refillHalf)(int16_t *)) {
    configureSAI();                              // SAI up, but TX not yet enabled
    i2s_ring = ring; i2s_ring_frames = nFrames; i2s_refill = refillHalf; i2s_blocks = 0;
    i2s_dma.begin();
    if (i2s_dma.TCD == 0) return false;          // no channel free
    i2s_dma.sourceBuffer((const uint16_t *)ring, (unsigned int)(nFrames * 2u * sizeof(int16_t)));
    i2s_dma.destination(*(volatile uint16_t *)&SAI1_TDR0);
    i2s_dma.interruptAtHalf();
    i2s_dma.interruptAtCompletion();
    i2s_dma.triggerAtHardwareEvent(DMAMUX_SOURCE_SAI1_TX);
    i2s_dma.attachInterrupt(i2s_dma_isr);
    i2s_dma.enable();
    hw->tcsr = SAI_TCSR_TE | SAI_TCSR_BCE | SAI_TCSR_FRDE;   // TX + bit clock + FIFO DMA request
    return true;
}

uint32_t I2SClass::dmaBlockCount() const { return i2s_blocks; }
```
